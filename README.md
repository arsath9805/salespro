\# SalesPro



A sales analytics and forecasting platform built using React, Firebase, and Python.



\## Features



\- Product Management (CRUD)

\- Search, Filter, Sorting, Pagination

\- Inventory Tracking

\- Sales Analytics Dashboard

\- Revenue \& Profit Analysis

\- Sales Forecasting

\- Notification System



\## Tech Stack



\### Frontend

\- React.js

\- JavaScript

\- CSS3

\- Plotly.js



\### Backend

\- Python

\- Flask



\### Database

\- Firebase Firestore



\## Modules



\### Products

Manage products, categories, inventory, and stock levels.



\### Analytics

Visualize revenue trends, top-selling products, customer insights, and inventory metrics.



\### Prediction

Predict future sales and generate business insights.



\### Notifications

Receive alerts for low stock, restocking, and new orders.
